과학자들은 인공지능을 사용하여 강력한 약물 내성의 '슈퍼박테리아'를 골라 죽이는 '슈퍼 항생제'를 찾았습니다. 이 항생제는 인공지능을 사용하여 슈퍼박테리아의 특정 단백질을 식별하고, 이 단백질을 표적으로 하는 새로운 항생제를 개발했습니다. 이 항생제는 아직 동물실험 단계에 있지만, 향후 슈퍼박테리아 감염을 치료하는 데 큰 도움이 될 것으로 기대됩니다.
